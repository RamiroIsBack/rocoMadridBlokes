Before creating any new file or making any change to existing files, always ask the user for confirmation first. Describe what you intend to do and wait for explicit approval before proceeding.

## Project Overview

React website for https://rocomadrid.com/blokes hosted on WordPress (SiteGround).

Content is sourced from a Google Form linked to Google Sheets, containing:
- 1–3 pictures per entry
- A 300-character text string
- A 20-character text string
- A toggle/selector with 4 options

## Stack & Hosting

- **Frontend**: React (Vite or CRA)
- **Hosting**: WordPress on SiteGround
- **Data source**: Google Forms → Google Sheets → Google Sheets API
- **Media**: Hosted on WordPress/SiteGround
